古着攻略小程序、微信小商店功能、个人闲置交易功能、留言板功能
该小程序是是一款为古着服务的小程序，收录了热门城市各地古着店信息，内含留言发布、个人闲置 交易、商家小商店等功能，代码中的服务器地址我全部替换成了url。
因为是新手第一次接触小程序，所以语法对新人很友好，大佬们轻点喷，谢谢
